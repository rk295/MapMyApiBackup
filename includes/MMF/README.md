mapmyfitness-php-sdk
====================

Community Supported MapMyFitness PHP SDK

Overview
--------------------

This is a basic php sdk for MapMyFitness, it currently uses oauth 1 and has the following supported api calls.

Visit www.mapmyapi.com

- Authentication
- User Information
- Workouts
- Courses
- Routes
